# Resume Screening AI

An NLP-based resume screening system that matches resumes with job
descriptions using TF-IDF and cosine similarity.

## Features
- PDF resume upload
- Job description matching
- Resume match score
- Skill gap analysis

## Tech Stack
- Python
- NLP (TF-IDF)
- Scikit-learn
- Google Colab

## How It Works
1. Upload resume (PDF)
2. Enter job description
3. System calculates similarity score
4. Displays missing skills

## Results
- Partial match: 20–30%
- Strong match: 40–60%

## Platform
Developed and tested on Google Colab
